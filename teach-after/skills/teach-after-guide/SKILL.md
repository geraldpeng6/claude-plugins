# 事后复盘教学指南

当用户完成 bug 修复或功能实现后，需要进行系统性复盘时，加载此技能。

## 快速开始

使用命令：`/teach-after:teach-after [复盘主题]`

## 适用场景

- Bug 修复后的复盘
- 功能实现后的总结
- 技术难题解决后的经验提炼

## 复盘流程

1. **重建时间线** - 梳理完整过程
2. **分析失败原因** - 理解失败尝试
3. **解释成功原因** - 说明最终方案
4. **原理深入讲解** - 抽象核心概念
5. **总结可迁移经验** - 提炼可复用模式
6. **后续优化建议** - 改进方向（可选）

## 使用示例

```
/teach-after:teach-after 登录失败 bug
```
