# 超级提示词指南

当用户需要深度思考、创意激发或问题分析时，加载此技能。

## 可用命令

- `/super-prompts:think-like` - 像天才一样思考
- `/super-prompts:ancient-modern` - 古今对话
- `/super-prompts:future-ai` - 未来AI视角
- `/super-prompts:hidden-truth` - 隐藏真相
- `/super-prompts:landing-page` - 落地页设计
- `/super-prompts:master-skill` - 大师技能
- `/super-prompts:multiverse` - 多元宇宙
- `/super-prompts:super-ai` - 超级AI

## 使用示例

```
/super-prompts:think-like 爱因斯坦
```
